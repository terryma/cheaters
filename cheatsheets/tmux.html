Prefix key: ⌃-Z

Window Management
Shortcut (-prefix) Description
⌥-N Create new window
⌥-H Go to left window
⌥-L Go to right window
⌥-; Go to last window
⌥-R Rename window
Pane Management
Shortcut (+prefix) Description
⌃-V Split pane vertically
⌃-N Split pane horizontally
⌃-H Go to left pane
⌃-J Go to below pane
⌃-K Go to above pane
⌃-L Go to right pane
⌃-O Rotate pane
⌃-P Switch to next layout
⌃-B Break pane into its own window
←↑→↓ Resize panes
Session Management
Shortcut (+prefix) Description
⌃-D Detach client
. Rename session
; Command prompt
Copy Mode
Shortcut (+prefix) Description
⌃-␣ Start copy mode
⌃-C Copy tmux buffer to system clipboard
⌃-X Paste tmux buffer
Misc
Shortcut (+prefix) Description
⌃-R Reload config
/ List key bindings
