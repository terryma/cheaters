Insert Mode
Shortcut Description
⌃-Q Quoted insert
⌃-W Delete previous word
⌃-E Go to end of line
⌃-R Regex history search backwards
⌃-T Set mark
⌃-Y Copy the region from the cursor to the mark to the clipboard
⌃-U Delete from cursor to the beginning of line
⌃-I Alias to TAB
⌃-O Delete from cursor to the end of line
⌃-P TODO
⌃-[ Alias to ESC
⌃-] TODO
⌃-\ TODO
⌃-A Go to beginning of line
⌃-S Regex history search forwards
⌃-D Delete next word
⌃-F Go up in history
⌃-G Go down in history
⌃-H Move one word left
⌃-J Move one character left
⌃-K Move one character right
⌃-L Move one word right
⌃-Z tmux prefix key
⌃-X Deletes the character under cursor
⌃-C Send SIGINT to foreground process
⌃-V Paste from clipboard
⌃-B TODO
⌃-N Clear screen
⌃-M Alias to RETURN
⌃-/ Undo
⌃-␣ Copy the current line to the clipboard
